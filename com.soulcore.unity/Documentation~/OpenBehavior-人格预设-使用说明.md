# OpenBehavior 人格预设使用说明

本文用于说明 `OpenBehaviorDriver` 的“人格倾向预设面板”与快速模板的推荐使用方式。

## 1. 快速上手

1. 选中场景中的 `OpenBehaviorDriver`。
2. 在 Inspector 的 `人格倾向预设面板` 中：
   - 可直接点击快速模板（社牛型/社恐型/高冷型/热心肠）。
   - 或先指定 `当前预设` 后点击“应用预设到当前组件”。
3. 点击快速模板后会自动：
   - 在 `Assets/SoulCore/Presets/` 生成对应 `.asset`；
   - 绑定到 `当前预设`；
   - 应用到当前组件参数。

## 2. 关键参数含义

- `目标选择人格权重`：人格信号对目标打分的影响强度。
- `语气人格权重`：人格信号对话术前缀/语气的影响强度。
- `话题人格权重`：人格信号对话题池偏好的影响强度。
- `任务期间人格权重覆盖`：任务 NPC 命中时，对上述权重乘任务期乘子。
- `紧急任务额外人格权重覆盖`：紧急任务命中时，在任务期基础上再乘一层额外乘子。

## 3. 推荐调参流程

1. 先选模板作为起点（例如“社恐型”）。
2. 打开：
   - `任务约束诊断日志`
   - `诊断附带人格生效权重`
3. 观察日志中的：
   - `isTaskNpc / isUrgentTaskNpc`
   - `effW(target/style/topic)`
4. 先调任务期乘子，再调紧急额外乘子，最后微调基础权重。

## 4. 常见问题

- **点击应用提示未选择预设**
  - 说明 `当前预设` 为空或缺失，请先选择预设资产；
  - 可使用“一键创建默认预设并应用”。

- **重开场景后参数不一致**
  - 推荐使用快速模板（会自动落盘为资产）；
  - 或手动创建并绑定预设资产，不要只做临时调参。

- **忙碌静默没生效**
  - 检查任务命中条件：标签/图层/直指定/自定义判定；
  - 检查是否命中紧急任务（日志 `isUrgentTaskNpc`）。
